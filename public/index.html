NC News API

A RESTful API for accessing news articles, topics, users, and comments.

  • Articles
    • GET /api/articles
    • GET /api/articles/:article_id
    • PATCH /api/articles/:article_id
    • GET /api/articles/:article_id/comments
    • POST /api/articles/:article_id/comments
  • Topics
    • GET /api/topics
  • Users
    • GET /api/users
  • Comments
    • GET /api/articles/:article_id/comments
    • POST /api/articles/:article_id/comments

GET /api/articles

Returns an array of all the article objects

  • Can filter by topic: ?topic=coding
  • Can sort by any valid column: ?sort_by= created_at
  • Can order: ?order=asc

Example Response

                    [
                    {
                    "title": "Seafood substitutions are increasing",
                    "topic": "cooking",
                    "author": "weegembump",
                    "created_at": "2018-05-30T15:59:13.341Z",
                    "votes": 0,
                    "comment_count": 6
                    }
                    ]

GET /api/articles/:article_id

Returns an article object by its ID

Example Response

                    {
                    "article_id": 1,
                    "title": "Living in the shadow of a greatness",
                    "topic": "mitch",
                    "author": "butter_bridge",
                    "body": "I find this existence challenging",
                    "created_at": "2018-11-15T12:21:54.171Z",
                    "votes": 100,
                    "comment_count": 13
                    }

PATCH /api/articles/:article_id

Increments or decrements the vote count of an article

Example Request

                    {
                    "inc_votes": 1
                    }

Example Response

                    {
                    "article_id": 1,
                    "title": "Living in the shadow of a greatness",
                    "topic": "mitch",
                    "author": "butter_bridge",
                    "body": "I find this existence challenging",
                    "created_at": "2018-11-15T12:21:54.171Z",
                    "votes": 101,
                    "comment_count": 13
                    }

GET /api/articles/:article_id/comments

Returns an array of comments for a specific article

Example Response

                    [
                    {
                    "comment_id": 1,
                    "author": "butter_bridge",
                    "body": "I love cheese",
                    "created_at": "2018-11-15T12:21:54.171Z",
                    "votes": 16
                    }
                    ]

POST /api/articles/:article_id/comments

Adds a new comment to an article

Example Request

                    {
                    "username": "butter_bridge",
                    "body": "I love cheese"
                    }

Example Response

                    {
                    "comment_id": 1,
                    "author": "butter_bridge",
                    "body": "I love cheese",
                    "created_at": "2018-11-15T12:21:54.171Z",
                    "votes": 0
                    }

GET /api/topics

Returns an array of all topics

Example Response

                    [
                    {
                    "slug": "coding",
                    "description": "Code is love, code is life"
                    }
                    ]

GET /api/users

Returns an array of all users

Example Response

                    [
                    {
                    "username": "butter_bridge",
                    "avatar_url": "https://avatars2.githubusercontent.com/u/24604688?s=400&v=4",
                    "name": "jonny"
                    }
                    ]
